About Us

YouTube Video Summarization Project

This project aims to provide a tool for summarizing YouTube video transcripts. Users can paste a YouTube video link, and the application will generate a summary of the video's content based on its transcript. The summaries can be helpful for quickly understanding the key points of a video or for skimming through lengthy content.

This application is developed using Python and Flask for the backend and HTML/CSS for the frontend. It utilizes the YouTube Transcript API to fetch the transcript of the provided video link and then applies text summarization techniques to generate concise summaries.
